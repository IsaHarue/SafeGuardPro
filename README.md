# 🛡️ SafeGuardPro

O **SafeGuardPro** é um aplicativo Android desenvolvido para auxiliar no gerenciamento de Equipamentos de Proteção Individual (EPIs), permitindo o controle de funcionários, equipamentos e entregas.

## 📱 Funcionalidades

- Login de usuários
- Cadastro de funcionários
- Cadastro de EPIs
- Controle de entregas de equipamentos
- Consulta de EPIs por funcionário

## 🛠 Tecnologias

- Kotlin
- Android Studio
- MVVM
- Retrofit
- Room Database
- Coroutines
- LiveData
- Material Design

## ▶️ Como executar

1. Clone este repositório.
2. Abra o projeto no Android Studio.
3. Aguarde a sincronização das dependências.
4. Execute o aplicativo em um dispositivo ou emulador Android.

## 📚 Sobre o projeto

Este projeto foi desenvolvido em equipe durante o curso Técnico em Análise e Desenvolvimento de Sistemas do **SENAI**, com orientação de um professor tutor. O objetivo foi aplicar conhecimentos de desenvolvimento Android utilizando Kotlin, arquitetura MVVM e boas práticas de desenvolvimento mobile.
